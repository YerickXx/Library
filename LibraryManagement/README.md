
# 📚 Library Management System

A Java-based desktop application designed to streamline library operations, including book inventory management, member registration, and loan transaction tracking.

## 🚀 Core Functionalities
* **Inventory Control:** Add, update, and remove books from the system.
* **Loan Management:** Process check-outs and check-ins with real-time availability updates.
* **Search Engine:** Filter books by title, author, or ISBN.
* **Member Records:** Manage user profiles and their borrowing history.

## 🛠️ Technical Highlights
* **OOP Principles:** Extensive use of Encapsulation, Inheritance, and Polymorphism.
* **Data Persistence:** Implementation of data structures (ArrayList/HashMap) to manage runtime information.
* **Robust Validations:** Error handling for duplicate entries and overdue returns.

## 📂 Architecture
The project follows a modular structure to ensure maintainability:
- `Models`: Entity definitions (Book, User, Loan).
- `Controller`: Business logic and transaction rules.
- `View`: Swing-based GUI for administrative tasks.
